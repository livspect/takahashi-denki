# 撮影してほしい画像リスト（写真撮影指示書）

現在サイトで使っている画像の多くは海外のフリー素材です。**御社で撮影した実際の写真**に差し替えると、日本＝当然・完全オリジナルで、信頼感が一気に上がります。
下記の優先度順に、撮れるものから送ってください（スマホ撮影でOK）。

## 撮影の共通ルール
- **明るく・水平に**：日中の自然光が理想。傾かないように。
- **向き・比率**：原則 **横位置（16:9 くらい）**。一部「縦位置」と明記したものだけ縦で。
- **高解像度**：長辺 2000px 以上（スマホの標準画質でOK。加工はこちらで行います）。
- **整理整頓**：背景に不要物・個人情報（書類・ナンバープレート等）が写らないように。
- **人物**：顔が写る場合は本人の掲載許可を取得。後ろ姿・手元中心でも可。
- **ブランド**：パナソニック等のロゴ写り込みはOK（むしろ「パナソニックの店」が伝わって良い）。
- ファイル名は自由。**「どの用途か」だけメモ**を添えてください。

## ＋ 随時追加（microCMSから自分で登録する画像）
ブログ記事・施工事例は管理画面（microCMS）から画像を登録します。記事/事例を作るたびに用意してください。

| 用途 | 内容 | 向き |
|---|---|---|
| ブログ記事のアイキャッチ | 記事内容に合った1枚 | 横 16:9 |
| 施工事例のサムネイル | 施工後（または施工前後）の写真 | 横 16:9 |

> 施工事例は「施工前→施工後」の2枚があると説得力が出ます（現状は1枚表示。ご希望なら前後比較レイアウトも作れます）。
